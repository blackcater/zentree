# @acme-ai/tsconfig

`@acme-ai/tsconfig` provides shared TypeScript configurations for the
Acme monorepo. It includes base configurations with strict type checking,
Node.js target settings, and React-specific presets.

**Configurations:**

- `base.json` - Base configuration with strict mode
- `nextjs.json` - Next.js specific settings
- `react-native.json` - React Native settings
- `node18.json` - Node.js 18+ target

**Usage:** Reference in `tsconfig.json` extends field
