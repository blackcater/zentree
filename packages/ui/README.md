# @acme-ai/ui

`@acme-ai/ui` is a cross-platform React component library that provides
consistent UI primitives for all Acme applications. Built on Radix UI with
custom theming.

**Key Features:**

- Design system aligned with Acme's UI/UX spec
- Radix UI primitives for accessibility
- Dark/light mode theming
- Cross-platform consistency (Desktop, Web, Mobile)
- Storybook documentation

**Components:** Button, Input, Dialog, Dropdown, Tabs, Tree, and more
