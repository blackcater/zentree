Acme 是一款面向代码智能体的桌面应用，作为 Codex App 的替代方案，
提供多保险库项目管理、多智能体支持以及本地优先的数据存储。
它为开发者提供了统一的工作空间，支持并行运行多个代码智能体。

## 功能

- 本地优先：所有数据本地存储，无云依赖
- 多智能体：支持并行运行多个代码智能体（Claude Code、Codex、
  Acme Code、ACP 兼容）
