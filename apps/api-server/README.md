# Acme API Server

AES (Acme Enterprise Server) is a RESTful API server built with Elysia,
providing enterprise-grade APIs for Acme. It handles authentication, billing,
team management, and so on.

**Key Features:**

- RESTful API with OpenAPI documentation
- OAuth 2.0 authentication and API key management
- Team management and role-based access control
- Usage tracking and cost attribution

**Tech Stack:** Elysia + TypeScript
