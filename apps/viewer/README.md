# Acme Viewer

AVA (Acme Viewer Application) is a lightweight web viewer for sharing
Acme threads. It allows anyone to view shared conversation threads without
requiring an Acme installation, making it ideal for code reviews and
collaboration.

**Key Features:**

- Read-only thread viewing
- Syntax-highlighted code blocks
- Message reactions and annotations
- Shareable links with access control

**Tech Stack:** Next.js + TypeScript
